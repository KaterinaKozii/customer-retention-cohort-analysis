**Customer Retention Cohort Analysis**

**Опис проєкту**

Метою цього проєкту є аналіз утримання користувачів (Customer Retention) та оцінка ефективності різних каналів залучення клієнтів.
У межах аналізу було досліджено поведінку користувачів за когортами, сформованими за джерелом залучення (Organic та Promo), щоб визначити, які сегменти забезпечують вищу довгострокову цінність для бізнесу.

**Бізнес-завдання**

Компанія прагнула зрозуміти:
- Який канал залучення користувачів демонструє кращий рівень утримання;
- Як змінюється активність користувачів протягом часу;
- Чи виправдовують себе промоакції з точки зору довгострокового утримання клієнтів;
- Які канали доцільно масштабувати для підвищення якості користувацької бази.
  
**Використані інструменти**
- Google Sheets
- Cohort Analysis
- Retention Analysis
- Business Analytics
- Data Visualization
  
**Етапи аналізу**

1. Підготовка та перевірка даних.
2. Формування когорт користувачів.
3. Розрахунок показників утримання для кожної когорти.
4. Порівняння органічного та промо-трафіку.
5. Аналіз динаміки утримання користувачів у часі.
6. Формування бізнес-висновків та рекомендацій.
   
**Основні результати**
- Органічно залучені користувачі продемонстрували більш стабільний рівень утримання.
- Промоактивності забезпечують більший початковий обсяг користувачів, але характеризуються швидшим відтоком.
- Якість залучення користувачів виявилася важливішою за їх кількість.
- Для покращення довгострокових показників доцільно оптимізувати промо-кампанії та посилити фокус на якісних каналах залучення.

**Бізнес-рекомендації**
- Збільшити інвестиції в канали, що забезпечують вищий рівень утримання;
- Переглянути механіку промоакцій для підвищення якості залучених користувачів;
- Регулярно проводити когортний аналіз для моніторингу змін у поведінці клієнтів;
- Використовувати retention-метрики як один із ключових показників ефективності маркетингових активностей.
